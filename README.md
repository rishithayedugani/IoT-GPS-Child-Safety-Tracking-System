# IoT-Based GPS Child Safety and Tracking System with Geofencing Alert

## 📌 Overview
This project is an IoT-based child safety system that uses GPS and geofencing technology to monitor a child's location in real time. The system continuously tracks the child's position and sends an alert whenever the child moves outside a predefined safe zone.

## 🚀 Features
- Real-time GPS location tracking
- Geofencing for safe zone monitoring
- Instant alert notifications
- Remote monitoring through IoT platform
- Low-cost and portable solution

## 🛠 Technologies Used
- ESP32 / NodeMCU (ESP8266)
- GPS Module (Neo-6M)
- Arduino IDE
- Wi-Fi Communication
- IoT Cloud Platforms (Blynk / ThingSpeak / Firebase)

## ⚙️ Hardware Components
- ESP32 / NodeMCU Controller
- GPS Neo-6M Module
- Buzzer
- Battery / Power Supply

## 🔄 Working Principle
1. The GPS module continuously tracks the child's location.
2. The ESP32 receives the GPS coordinates.
3. The system compares the current location with predefined geofence coordinates.
4. If the child remains inside the safe zone, the system operates normally.
5. If the child leaves the safe zone, an alert is sent to parents through a mobile application or SMS.
6. The live location can be monitored through an IoT platform.
